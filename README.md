# Predictive Modeling of Medical Costs

## Overview
This project uses machine learning to predict medical insurance costs based on individual characteristics such as age, BMI, smoking status, and more.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## Project Steps
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature selection and engineering
4. Model training (Linear Regression / Ridge / Lasso)
5. Model evaluation using RMSE, MAE, and R²

## Key Insights
- Smoking status is the strongest predictor of medical costs
- Higher BMI is associated with increased insurance charges
- Age has a positive correlation with medical expenses

## Results
The model achieved strong predictive performance and can be used to estimate healthcare costs based on patient characteristics.
